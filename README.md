# TL All-in-One Suite 🚀

[Português](#português-🇧🇷) | [English](#english-🇺🇸)

---

## Português 🇧🇷

Uma suite de scripts unificada e poderosa, desenvolvida para operadores e gestores de Sort Centers da Amazon. Consolida múltiplas ferramentas em um único painel de alta performance com interface moderna em glassmorphism.

### 📦 Módulos Incluídos

-   **📈 TPH Chart (V5):** Painel de throughput em tempo real com cálculo dinâmico de necessidade.
-   **👥 Dynamic Container Builder Ranking:** Ranking de produtividade de associados com busca e auto-scroll.
-   **🚛 VSM Map / Dock View:** Mapa interativo das docas (IB, OB, Reversa) e integração com YMS.
-   **🔍 VRID Info & CPT Tracker:** Detalhes técnicos de VRIDs, análise de volumetria em CuFt, identificação de X-Dock e exportação para Excel.
-   **📅 CPT Tracker:** Monitoramento de tempos críticos (CPT) com alertas de urgência.

### 🛠️ Principais Funcionalidades

-   **Interface Moderna:** Design elegante com modo escuro nativo.
-   **Multilíngue:** Suporte completo para Inglês e Português (PT-BR).
-   **Performance:** Busca de dados paralela e renderização otimizada.
-   **Customização:** Configurações persistentes de Node ID, metas e pausas.

### ⚠️ Compatibilidade
Atualmente, o script é compatível **apenas com o navegador Firefox**. O suporte para Chrome e outros navegadores baseados em Chromium está em desenvolvimento.

### 🚀 Instalação

1.  Instale a extensão **Tampermonkey** no seu **Firefox** ([Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)).
2.  Clique [aqui](https://raw.githubusercontent.com/Soakll/sort-center-tools/main/tl-suite.user.js) para instalar o script.
3.  Recarregue as páginas do SSP (Site Status), YMS, etc.

## English 🇺🇸

A powerful and unified userscript suite designed for Amazon Sort Center operators and managers. It streamlines multiple tools into a single, high-performance dashboard with a modern glassmorphism interface.

### 📦 Included Modules

-   **📈 TPH Chart (V5):** Real-time throughput dashboard with dynamic "Need" calculation.
-   **👥 Dynamic Container Builder Ranking:** Associate productivity tracking and ranking with auto-scroll.
-   **🚛 VSM Map / Dock View:** Interactive visual map of yard/dock sectors (IB, OB, Reversa).
-   **🔍 VRID Info & CPT Tracker:** Detailed VRID analytics, CuFt volumetric calculation, X-Dock identification, and Excel export.
-   **📅 CPT Tracker:** Critical Pull Time monitoring with visual urgency alerts.

### 🛠️ Key Features

-   **Modern UI:** Sleek glassmorphism design with native dark mode.
-   **Multi-language:** Full support for English and Portuguese (PT-BR).
-   **Performance:** Parallelized data fetching and optimized rendering.
-   **Customization:** Persisted settings for Node ID, goals, and breaks.

### ⚠️ Compatibility
Currently, the script is compatible **only with the Firefox browser**. Support for Chrome and other Chromium-based browsers is under development.

### 🚀 Installation

1.  Install the **Tampermonkey** extension for **Firefox** ([Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)).
2.  Click [here](https://raw.githubusercontent.com/Soakll/sort-center-tools/main/tl-suite.user.js) to install the script.
3.  Refresh your Amazon Sort Center pages (Site Status, YMS, etc.).
